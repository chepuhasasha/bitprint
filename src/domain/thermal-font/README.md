# Thermal Font Glyphs

Глифы хранятся отдельно по файлам в `glyphs/*.glyph.txt`.

## Формат файла

- Каждая строка файла = одна строка матрицы.
- `#` = закрашенный пиксель.
- ` ` (пробел) = пустой пиксель.
- Пустые строки тоже значимы и сохраняются как есть.

## Имена файлов

Формат имени:

`<HEX>__<TOKEN>[__SCRIPT].glyph.txt`

Примеры:

- `0041__A__latin.glyph.txt`
- `0416__Ж__cyrillic.glyph.txt`
- `0020__space.glyph.txt`
- `0025__percent.glyph.txt`

`HEX` однозначно задает символ (Unicode codepoint).
`TOKEN` нужен для удобства и безопасен для URL/Windows.
`SCRIPT` (если указан) помогает различать похожие символы, например латиницу и кириллицу.
